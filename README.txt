Az általam húzott tételek alapján az az ötletem támadt, hogy egy XML to CSV Convertert készítek.
Amelynek az két fő funkciója az lesz, hogy elsősorban beolvas egy adott Xml-t majd azt átalakítja
olyan formába amely már könnyedén kiírható CSV formátumba is. Ezután megtörténik a kiírás Csv-be.
Véleményem szerint ez egy hasznos funkció lehet mivel elképzelhető olyan életszituáció amikor
Xml fájlt Csv-be kell konvertálni. Emellett a timer-t egy óraként szeretném felhasználni az alkalmazásban.
Az alkalmazásom felhasználói felületét könnyen kezelhetőre terveztem, színvak barát színösszeállítással és
ezt kiegészítve egy súgóval.
